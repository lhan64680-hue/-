## 实现决策

- “素材备份”作为独立顶部标签页，顺序为：`项目库 / 素材备份 / 素材库 / 素材管理中心 / 报表 / 任务`。
- 无项目打开时允许浏览页面，但不允许启动备份；完成后需要归档到当前项目素材库。
- 默认不启用哈希校验；用户可选择大小校验、SHA-256 或 MD5。
- 第一版级联只做“源到主目标，再由主目标到多个二级目标”，不做任意多级瀑布链。
- 多目的地直接拷贝必须为每个目标独立 worker，避免慢盘拖住所有目标。
- 归档路径复用现有 `ImportService::importDirectory`，让素材库扫描、媒体解析、缩略图和素材管理中心同步沿用当前链路。
- 每个成功目标写入 `CineVaultBackupLog.json`，记录源、目标、文件数、字节数、校验模式、错误、开始/完成时间。
