# AI-DATA 拆分说明

## 拆分结果

从影资管家 v0.1.143 起，“素材备份”不再集成在影资管家内。备份能力已迁移到独立项目：

```text
G:\data\app\AI-DATA
```

AI-DATA 独立维护自己的构建系统、领域模型、复制服务、任务队列、界面与配置存储，不链接影资管家源码，也不读写影资管家项目数据库。

## 影资管家保留能力

- 项目库
- 素材库及左侧栏“添加素材源”
- 素材管理中心
- 报表
- 任务
- 使用反馈

影资管家不再提供“素材备份”或独立“导入”页面。需要向当前项目添加素材时，在素材库左侧栏点击“添加素材源”，继续复用项目扫描与索引能力。

## 已移除范围

- 素材备份 QML 工作台和导航标识
- 独立导入页面、导航、路由和页面专属 ViewModel
- 素材备份 ViewModel、列表模型与应用服务
- 备份计划、复制引擎、校验和磁盘弹出实现
- 备份队列配置项与领域实体
- 备份相关构建清单和单元测试

历史版本中已经保存的素材备份队列配置不会再被影资管家读取。
