# AI-DATA 拆分说明

## 拆分结果

从影资管家 v0.1.143 起，“素材备份”不再集成在影资管家内。备份能力已迁移到独立项目：

```text
G:\data\app\AI-DATA
```

AI-DATA 独立维护自己的构建系统、领域模型、复制服务、任务队列、界面与配置存储，不链接影资管家源码，也不读写影资管家项目数据库。

## 影资管家保留能力

- 项目库
- 导入素材源
- 素材库
- 素材管理中心
- 报表
- 任务
- 使用反馈

原“素材备份”导航位置恢复为“导入”，用于添加素材源并建立项目索引，不包含多目的地复制、级联复制或备份校验功能。

## 已移除范围

- 素材备份 QML 工作台和导航标识
- 素材备份 ViewModel、列表模型与应用服务
- 备份计划、复制引擎、校验和磁盘弹出实现
- 备份队列配置项与领域实体
- 备份相关构建清单和单元测试

历史版本中已经保存的素材备份队列配置不会再被影资管家读取。
