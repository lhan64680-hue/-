# 002 真实导入闭环测试清单

## 启动步骤

1. 打开本轮真实工作流发布目录 `output/vX.Y.Z/`
2. 双击 `CineVault.exe`
3. 等待主窗口打开

## 点击路径

1. 点击 `添加素材源`，此时尚未打开项目，应看到“请先创建或打开项目。”提示
2. 点击 `新建项目`
3. 输入项目名称，选择项目保存目录
4. 确认项目目录内生成 `project.cvdb`
5. 点击 `添加素材源`
6. 选择一个包含少量真实文件的目录
7. 等待扫描任务完成
8. 点击左侧素材源卡片
9. 切到 `素材库`，在大图卡片和技术表格间切换
10. 在顶部搜索框输入文件名片段
11. 点击任意素材卡片或表格行
12. 切到 `任务` 工作区查看扫描记录

## 预期结果

1. 未打开项目时添加素材源会显示中文提示，不崩溃
2. 新建项目会创建项目目录和 `project.cvdb`
3. 添加素材源后任务栏显示扫描进度
4. 扫描完成后素材源栏显示文件数、容量、视频数和警告数
5. 素材库显示真实扫描到的文件
6. 搜索框能过滤素材列表
7. 点击素材源后检查器显示素材源统计
8. 点击素材后检查器显示文件路径、类型、大小、修改时间和可读状态
9. 顶部栏、卡片、检查器、表格和任务列表中文文本不越框

## 当前不包含

- FFmpeg 媒体元数据提取
- 缩略图生成
- 拖拽导入
- PDF 报表导出
