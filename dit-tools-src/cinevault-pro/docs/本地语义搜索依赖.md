# 本地语义搜索依赖

## 锁定原则

- 应用运行时禁止联网下载模型、运行库或向量索引依赖。
- 构建资产以 `cmake/local-search-dependencies.lock.json` 为机器可读来源，并逐文件校验 SHA-256 和大小。
- `dependencies.lock` 保留面向开发者的版本、许可证和环境说明。
- 依赖缓存必须位于源码目录之外，避免模型和二进制资产污染 Git 工作区。

## 准备构建缓存

```powershell
cmake `
  -DOUTPUT_ROOT="$env:LOCALAPPDATA\CineVault\build-dependencies" `
  -P cmake/PrepareLocalSearchDependencies.cmake
```

脚本只下载锁清单中的文件，使用临时 `.part` 文件写入并在摘要、大小都匹配后原子替换。重复执行会复用已经校验通过的文件。

当前脚本只准备并验证构建资产。ONNX Runtime、USearch 与模型的编译链接、部署和安装包集成在自然语言混合搜索模块中启用。

## 固定帧约束

`src/shared/SearchConfiguration.h` 将视觉采样策略固定为 `fixed_interval`。设置值必须是正整数；任何变化帧、关键帧、场景切换或自适应采样实现都不属于受支持策略。
