# 影资管家 Windows 构建与依赖约定

## 推荐安装组合

1. Visual Studio 2022 Build Tools 或更高版本
2. Windows SDK 10.0.22621.0 或更高版本
3. Qt 6.x MSVC 64-bit Kit
4. CMake
5. Ninja
6. FFmpeg development package（阶段 4 需要）

## 必要环境变量

- `QT_ROOT`
  - 指向 Qt MSVC Kit 根目录
  - 目录内必须存在 `lib/cmake/Qt6/Qt6Config.cmake`
  - 当前已验证可用路径示例：`C:\Qt\6.6.3\msvc2019_64`
- `FFMPEG_DEV_ROOT`
  - 指向 FFmpeg 开发包根目录
  - 目录内必须存在 `include/`、`lib/`、`bin/`

## 目录约定

- 仅命令行运行包：`G:\data\app\DIT\ffmpeg`
  - 该目录当前只有 `ffmpeg.exe`、`ffprobe.exe`
  - 不满足库级集成
- 推荐开发包目录：
  - `G:\data\app\DIT\ffmpeg-dev`
  - 或任意自定义目录，再通过 `FFMPEG_DEV_ROOT` 指定

## 构建命令

不启用 FFmpeg：

```powershell
powershell -ExecutionPolicy Bypass -File tool/build_windows.ps1
```

启用 FFmpeg：

```powershell
$env:FFMPEG_DEV_ROOT="G:\data\app\DIT\ffmpeg-dev"
powershell -ExecutionPolicy Bypass -File tool/build_windows.ps1 -EnableFfmpeg
```

## 当前阶段策略

- 阶段 1-3 功能必须在没有 FFmpeg 开发包的情况下可构建
- 阶段 4 媒体模块只有在提供 `FFMPEG_DEV_ROOT` 时才参与编译
- 里程碑 A 默认构建最小 GUI 壳体，不启用 FFmpeg
- 当前 Windows 构建脚本允许：
  - Visual Studio 2022 Build Tools 路径回退探测
  - Qt `msvc2022_64`
  - Qt `msvc2019_64`
