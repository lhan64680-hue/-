# 001 最小GUI首测清单

## 启动步骤

1. 在 Windows 本机定位到本轮产出的 `dist/vX.Y.Z/`
2. 双击 `CineVault.exe`
3. 等待主窗口打开

## 点击路径

1. 观察窗口标题是否为“影资管家”
2. 依次点击顶部工作区：`导入`、`素材库`、`检查/质检`、`报表`、`任务`
3. 回到 `导入` 工作区，点击：
   - `新建项目`
   - `打开项目`
   - `添加素材源`
4. 点击左侧素材源卡片
5. 切到 `素材库`，分别点击：
   - `大图卡片`
   - `技术表格`
   - 任意素材卡片或表格行

## 预期结果

1. 程序能启动，不闪退
2. 主窗口显示完整五区布局：
   - 顶部指挥台
   - 左侧素材源栏
   - 中部工作区
   - 右侧检查器
   - 底部任务时间线
3. 五个工作区切换时，中部内容会变化
4. `新建项目` / `打开项目` 点击后，顶部项目名称或导入页提示文案发生变化
5. 点击左侧素材源后，右侧检查器显示对应素材源信息
6. 点击素材卡片或表格行后，右侧检查器显示对应素材信息
7. 任务页能显示占位任务列表

## 如果失败应看到的现象

1. 若双击无反应或立刻退出：说明运行时依赖未部署完整
2. 若窗口能开但界面为空白：说明 QML 模块或 Qt 插件未正确部署
3. 若按钮能点但页面不切换：说明 `shellVm.currentWorkspace` 绑定异常
4. 若点击素材后检查器不刷新：说明最小联动逻辑未接通
