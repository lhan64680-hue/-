核心能力

围绕素材本身，而不是围绕文件夹打转

从现场收卡后的项目整理，到后期阶段的内容定位与报表交付，关键上下文始终留在同一套系统中。

01
库

项目库与素材源

每个项目独立管理素材路径、数据库和索引状态。原文件保持原位，项目结构、目录层级和离线状态都能被持续追踪。

  • 项目入口管理
  • 素材源扫描
  • 离线路径兼容
02
览

多格式统一预览

视频、音频、图片和文档统一浏览，卡片与技术表格两种视图自由切换。

03
析

逐帧视觉解析

按固定帧间隔提取画面，生成摘要、实体事实、OCR 和自定义多维度分析。

04
搜

全局混合检索

融合路径、日期、文件类型、结构化属性、全文与本地语义向量，快速找回目标素材。

05
任

任务与进度

集中查看当前批次、成功、失败和排队数量，逐条任务拥有明确的进度与错误信息。

06
报

可预览的专业报表

在导出前直接检查封面摘要、项目表头、素材统计、缩略图、技术元数据和文件夹树，让交付结果可控。

  • PDF 分页预览
  • 按素材源导出
  • 技术元数据汇总
工作流程

四步建立可持续的素材资产

先建立真实目录与项目的映射，再逐步补充预览、视觉理解和交付信息。

  1. 01
    建立项目

    创建或打开项目，接入一个或多个素材源。

  2. 02
    扫描与预览

    生成目录层级、技术元数据和缩略图索引。

  3. 03
    视觉理解

    按固定间隔解析帧画面，沉淀可检索的结构化事实。

  4. 04
    检索与交付

    用自然语言找素材，预览并导出项目报表。
