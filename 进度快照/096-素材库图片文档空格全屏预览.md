## 具体修改的代码前后对比

### 1. 素材库空格全屏预览目标扩展

修改前：

```qml
function openSelectedAssetFullscreen() {
    if (viewModel && viewModel.selectedPreviewIsVideo) {
        fullscreenPlaybackBridge.openFullscreen()
    }
}
```

修改后：

```qml
function openSelectedAssetFullscreen() {
    if (!viewModel) {
        return
    }

    if (viewModel.selectedPreviewIsVideo) {
        fullscreenPlaybackBridge.openFullscreen()
    } else if (viewModel.selectedPreviewIsImage) {
        assetPreviewOverlay.openImage(viewModel.selectedAssetUrl, viewModel.selectedPreviewTitle)
    } else if (viewModel.selectedPreviewIsDocument) {
        assetPreviewOverlay.openDocument(viewModel.selectedAssetUrl, viewModel.selectedPreviewTitle)
    }
}
```

修改前：

修改后：

```cpp
if (suffix == QStringLiteral("pdf")) {
    PreviewPayload payload;
    payload.isPdf = true;
    return payload;
}
if (suffix == QStringLiteral("md")) {
    auto payload = readPlainTextPreview(path);
    payload.isMarkdown = payload.errorMessage.isEmpty();
    return payload;
}
if (suffix == QStringLiteral("docx")) {
    return buildDocxPreview(path);
}
if (suffix == QStringLiteral("xlsx")) {
    return buildXlsxPreview(path);
}
if (suffix == QStringLiteral("pptx")) {
    return buildPptxPreview(path);
}
```
