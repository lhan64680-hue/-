## 端口与隔离检查

部署前检查了远端 `3413–3499` 监听状态，没有发现占用，因此选择 `3413`。最终监听状态为：

```text
127.0.0.1:3413 -> uvicorn / cinevault-support-site.service
```

服务不会监听公网网卡，只通过 `dit.ee2x.cn` 的 Nginx 反向代理访问。原 `pond5-support-site.service` 保持 active，`https://pd.ee2x.cn` 在部署后仍返回 HTTP 200。

```powershell

```
