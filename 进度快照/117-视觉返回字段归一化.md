# 117-视觉返回字段归一化

日期：2026-06-29

## 已完成内容

- 已读取上一轮最新快照：`116-视觉返回解析器抽离.md`。
- 已创建阶段备份：`backup/v0.1.99`。
- 已完成模块二：字段归一化与轻量 schema 校验。
- `VisionResponseParser` 新增 `normalizeFrameAnalysis()` 和 `normalizeVideoSummary()`。
- 帧解析支持常见字段别名：`description`、`keywords`、`subjects`、`action`、`scene` 等会映射回现有 `caption/tags/objects/actions/setting`。
- 视频汇总支持常见字段别名：`overview`、`tags`、`locations` 等会映射回现有 `summary/keywords/scenes`。
- 列表字段支持字符串、数组、逗号/顿号/分号分隔字符串，以及 JSON 数组字符串。
- 归一化后如果业务字段全部为空，会返回明确错误，避免空结果被当成成功解析。
- `VisionApiClient` 已改为使用归一化后的业务对象。
- 已补充单测覆盖字段别名、字符串列表、嵌套 text/name 对象、空结果校验。
- 已更新 `docs/素材管理.md` 的视觉接口说明。
- 已新增模块溯源文档：`backup/v0.1.99/视觉返回字段归一化-前后对比.md`。

## 当前修改到哪个模块

当前完成模块：模块二，字段归一化与轻量 schema 校验。

本模块仍然要求模型返回可解析 JSON；CSV、纯文本、Markdown 表格等非 JSON 内容留到模块三做自动修复请求。

## 具体修改的代码前后对比

### 帧解析字段读取

修改前：

```cpp
VisionFrameAnalysis analysis;
analysis.caption = payload->value(QStringLiteral("caption")).toString().trimmed();
analysis.tags = jsonStringList(payload->value(QStringLiteral("tags")));
analysis.objects = jsonStringList(payload->value(QStringLiteral("objects")));
analysis.actions = payload->value(QStringLiteral("actions")).toString().trimmed();
analysis.setting = payload->value(QStringLiteral("setting")).toString().trimmed();
return analysis;
```

修改后：

```cpp
return VisionResponseParser::normalizeFrameAnalysis(*payload, errorMessage);
```

### 视频汇总字段读取

修改前：

```cpp
VisionVideoSummary summary;
summary.summary = payload->value(QStringLiteral("summary")).toString().trimmed();
summary.keywords = jsonStringList(payload->value(QStringLiteral("keywords")));
summary.scenes = jsonStringList(payload->value(QStringLiteral("scenes")));
return summary;
```

修改后：

```cpp
return VisionResponseParser::normalizeVideoSummary(*payload, errorMessage);
```
