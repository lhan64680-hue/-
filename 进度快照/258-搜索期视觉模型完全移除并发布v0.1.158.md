# 258 - 搜索期视觉模型完全移除并发布 v0.1.158

## 任务目标

按用户确认，将视觉模型完全移出搜索时流程；视觉模型只保留在素材导入/解析阶段，用来生成画面描述、实体、OCR、动作、场景和帧级事实。搜索阶段只使用本地规则、FTS5、结构化实体、BGE/USearch 和按需启动的本地 Qwen3 文本助手。

## 当前模块

本轮修改、测试和发布均已完成，版本为 `v0.1.158`。

## 已完成内容

### 1. 移除搜索时视觉复核

- `MaterialCenterViewModel` 不再接收或保存 `VisionApiClient`。
- 删除 `FrameRerankTaskResult`、候选帧缓存、在途任务和异步远程调用。
- 删除 `startFrameRerank` 与 `applyFrameRerank`。
- 帧搜索结果始终按本地混合搜索结果排序，不再上传前 8 个候选帧。
- 本地文本助手完成查询补充后直接执行第二轮本地搜索。

### 2. 删除搜索专用视觉 API

- 从 `VisionApiClient` 删除 `understandSearchQuery`。
- 从 `VisionApiClient` 删除 `rerankFrameCandidates`。
- 删除 `ModelFrameRerankScore`。
- 删除 `frameRerankResponseSchema` 和 `parseFrameRerankPayload`。
- 视觉客户端继续保留 `analyzeFrame`、`analyzeImage`、`summarizeVideo`、多维度分析和连通性测试等素材解析接口。

### 3. 清理设置和界面

- 删除 `frameRerankEnabled`。
- 删除 `allowSearchFrameUpload`。
- 删除已失去意义的 `localOnlySearch`；搜索现已固定为本地执行。
- 设置页只保留“按需使用内置轻量文本模型辅助理解查询”。
- 设置页明确显示：视觉接口只用于素材导入和解析，不参与搜索，也不会在搜索时发送候选帧。
- 快捷搜索状态从“视觉语言模型增强中”改为“内置文本模型增强中”。

### 4. 文档更新

- 更新 `内置轻量文本搜索助手完整方案.md`，把纯本地搜索确认为当前架构。
- 将 `视觉模型辅助模糊搜索方案.md` 标记为历史方案，防止旧设计被误认为当前实现。

## 关键代码前后对比

### MaterialCenter 构造依赖

修改前：

```cpp
MaterialCenterViewModel(...,
                        AppSettings *settings,
                        VisionApiClient *visionApiClient,
                        LocalSearchAssistantRuntime *runtime,
                        SearchAssistantClient *client,
                        QObject *parent);
```

修改后：

```cpp
MaterialCenterViewModel(...,
                        AppSettings *settings,
                        LocalSearchAssistantRuntime *runtime,
                        SearchAssistantClient *client,
                        QObject *parent);
```

搜索 ViewModel 已无法调用视觉接口。

### 文本模型完成后的链路

修改前：

```cpp
executeSearch(&*task.understanding);
startFrameRerank(m_lastParsedQuery);
```

修改后：

```cpp
executeSearch(&*task.understanding);
```

第二轮搜索完成后直接使用本地排序。

### 视觉客户端公开能力

修改前：公开远程查询理解、候选帧复核和素材分析 API。

修改后：仅保留素材分析 API；搜索理解由 `SearchAssistantClient` 在本机完成。

## 最终搜索链路

```text
用户文字
  -> 确定性日期/类型/实体解析
  -> FTS5 + 结构化事实 + BGE/USearch 首轮搜索
  -> 可靠性评估
  -> 必要时本地 Qwen3 0.6B 补充查询计划
  -> 第二轮本地混合搜索
  -> 本地排序并展示
```

搜索链路中不存在视觉接口或缩略图上传步骤。
