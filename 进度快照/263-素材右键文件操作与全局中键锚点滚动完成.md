# 263 - 素材右键文件操作与全局中键锚点滚动完成

## 任务目标

1. 快捷搜索结果增加“打开所在目录、复制文件路径”右键菜单及对应功能。
2. 素材管理中心和素材库全部素材卡片增加相同的文件操作菜单。
3. 所有显式滚动浏览页面支持鼠标中键按下显示锚点，并按指针相对锚点的距离持续滚动。
4. 完成全量测试、Release 真工作流构建和 `v0.1.163` 安装包。

## 当前模块

本轮三个功能模块以及构建验证均已完成。当前发布候选版本为 `v0.1.163`。

## 已完成内容

### 1. 快捷搜索右键文件操作

- 文件夹搜索结果支持：打开所在目录、复制文件夹绝对路径。
- 素材和逐帧搜索结果支持：打开素材所在目录、复制素材绝对路径。
- 右键时先同步当前结果标识，菜单操作不依赖旧选中项。
- 快捷搜索仍保留左键选择、双击打开和键盘操作。

### 2. 素材管理中心全部结果卡片

- 视觉帧结果卡片、文件夹结果卡片、素材结果卡片全部接入右键菜单。
- 素材文件操作统一按 `videoKey` 解析，普通素材列表、详情缓存和逐帧搜索结果都能回查真实素材路径。
- 路径不可用或目录打不开时，通过素材管理中心状态消息给出明确反馈。
- 复制到剪贴板的路径使用 Windows 原生分隔符，并归一化为完整绝对路径。

### 3. 素材库全部卡片

- 宫格卡片和列表行的原右键菜单均新增“复制文件路径”。
- 原“打开文件所在目录”统一为“打开所在目录”。
- 完整版与最小版素材库 ViewModel 都增加 `copyAssetPath(assetId)`，两种构建模式接口一致。
- 收藏、取消收藏和移除文件功能保持不变。

### 4. 全局中键锚点滚动

- `MiddleDragScrollHandler` 从直接拖拽位移改为带可见圆环十字锚点的持续滚动。
- 中键按下时记录锚点；相对位移方向决定滚动方向，位移越大速度越快。
- 锚点周围 10 像素为死区，避免轻微手抖触发滚动。
- 约 60 FPS 定时更新，单帧速度受上限约束；松开中键立即停止并回到合法边界。
- 同时支持纵向、横向以及两个不同滚动容器的联动。
- 素材库表格用一个处理器联动外层横向 `Flickable` 和内层纵向 `ListView`。
- 共识别 23 个显式 `ListView / GridView / Flickable / ScrollView`，接入 22 个处理器；少一个是素材库表格的双容器共用处理器，不是遗漏。
- 覆盖项目库、素材库、素材管理中心、报表、任务、反馈、设置、快捷搜索、素材源栏、详情栏、任务详情、时间线、文档预览和下拉列表。

## 关键代码前后对比

### 快捷搜索结果

修改前：只有左键选择和双击打开。

```qml
MouseArea {
    anchors.fill: parent
    onClicked: root.selectedFlatIndex = index
    onDoubleClicked: resultDelegate.activate(...)
}
```

修改后：增加独立菜单，并把当前卡片的 `videoKey` 直接交给文件操作接口。

```qml
ThemedMenuItem {
    text: "打开所在目录"
    onTriggered: materialCenterViewModel.openAssetFolder(resultDelegate.videoKeyValue)
}
ThemedMenuItem {
    text: "复制文件路径"
    onTriggered: materialCenterViewModel.copyAssetPath(resultDelegate.videoKeyValue)
}
```

### 路径复制

修改前：素材库只有打开目录，没有复制路径接口。

修改后：统一写入原生完整路径。

```cpp
clipboard->setText(
    QDir::toNativeSeparators(QFileInfo(asset.absolutePath).absoluteFilePath()));
```

### 中键滚动

修改前：中键位移直接换算为一次性内容偏移，没有可见锚点。

```qml
flickable.contentY = boundedContentY(startContentY - activeTranslation.y)
```

修改后：保存相对锚点距离，由定时器按距离计算连续滚动步长。

```qml
function stepForDistance(distance) {
    var magnitude = Math.abs(distance) - deadZone
    return magnitude <= 0
        ? 0
        : Math.sign(distance) * Math.min(maximumStep, magnitude * speedFactor)
}

property Timer autoScrollTimer: Timer {
    interval: 16
    running: handler.active
    onTriggered: handler.scrollStep()
}
```
