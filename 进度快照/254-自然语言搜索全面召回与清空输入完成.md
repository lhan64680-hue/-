## 关键代码前后对比

### 1. 查询解析重叠约束

修改前：

```cpp
const auto materials = matchingTerms(working, materialDictionary());
// “牛仔裤”同时产生 label=牛仔裤、materials=[牛仔]
```

修改后：

```cpp
auto materials = matchingTerms(working, materialDictionary());
if (!labels.isEmpty()) {
    removeTermsImpliedByLabel(&materials, labels.first());
}
```

### 2. 素材严格事实门禁

修改前：

```cpp
if (!assetsWithCompleteFacts.contains(hit.entityKey)) {
    ++result.excludedPartialCount;
    continue;
}
```

修改后：

```cpp
const bool structuredMatch = assetsWithCompleteFacts.contains(hit.entityKey)
    && factsMatchAllConstraints(...);
if (!structuredMatch && !incompleteTextEvidenceByAsset.contains(hit.entityKey)) {
    if (!assetsWithCompleteFacts.contains(hit.entityKey)) {
        ++result.excludedPartialCount;
    }
    continue;
}
```

### 3. 多词排序

修改前：只保留单个最佳词匹配分数。

修改后：统计全部查询词覆盖率，摘要/关键词完整覆盖时加入 `查询关键词完整覆盖` 原因并提高词法分数。

### 4. 清空输入

修改前：搜索框后直接排列项目筛选器。

修改后：

```qml
ActionButton {
    text: "清空输入"
    enabled: searchInput.text.length > 0
    onClicked: {
        shellVm.globalSearchText = ""
        searchInput.forceActiveFocus()
    }
}
```
