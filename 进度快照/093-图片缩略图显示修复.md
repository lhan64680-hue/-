## 具体修改的代码前后对比

### 1. ThumbnailEngine 图片固定取第 1 帧

修改前：

```cpp
auto normalizedRequest = request;
if (m_settings) {
    normalizedRequest.frameIndex = m_settings->thumbnailFrameIndex();
}
return m_adapter->generateThumbnail(normalizedRequest);
```

修改后：

```cpp
auto normalizedRequest = request;
if (normalizedRequest.assetType == AssetType::Image) {
    normalizedRequest.frameIndex = 1;
} else if (m_settings) {
    normalizedRequest.frameIndex = m_settings->thumbnailFrameIndex();
}
return m_adapter->generateThumbnail(normalizedRequest);
```

### 2. 素材列表图片路径兜底

修改前：

```cpp
"af.is_favorite, COALESCE(th.image_path, ''), COALESCE(mm.container, ''), COALESCE(mm.duration_ms, 0), "
```

修改后：

```cpp
"af.is_favorite, "
"CASE WHEN af.asset_type = %1 AND COALESCE(th.image_path, '') = '' THEN af.absolute_path ELSE COALESCE(th.image_path, '') END, "
"COALESCE(mm.container, ''), COALESCE(mm.duration_ms, 0), "
...
"WHERE 1 = 1").arg(static_cast<int>(AssetType::Image));
```

### 3. 素材检查器缩略图路径兜底

修改前：

```cpp
"af.is_favorite, mm.probe_status, mm.container, mm.duration_ms, mm.bit_rate, mm.error_message, th.image_path, mm.raw_json "
...
"WHERE af.id = ?"));
```

修改后：

```cpp
"af.is_favorite, mm.probe_status, mm.container, mm.duration_ms, mm.bit_rate, mm.error_message, "
"CASE WHEN af.asset_type = %1 AND COALESCE(th.image_path, '') = '' THEN af.absolute_path ELSE th.image_path END, "
"mm.raw_json "
...
"WHERE af.id = ?").arg(static_cast<int>(AssetType::Image)));
```
