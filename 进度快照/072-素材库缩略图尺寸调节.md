## 代码前后对比

### 卡片预览高度

修改前：

```qml
Rectangle {
    Layout.fillWidth: true
    Layout.preferredHeight: 146
}
```

修改后：

```qml
property int previewHeight: 146

Rectangle {
    Layout.fillWidth: true
    Layout.preferredHeight: previewHeight
}
```

### 网格尺寸与 Ctrl+滚轮

修改前：网格尺寸固定。

```qml
cellWidth: 240
cellHeight: 282
```

修改后：

```qml
property real thumbnailScale: 1.0
readonly property int gridPreviewHeight: Math.round(146 * thumbnailScale)
readonly property int gridCellWidth: gridCardWidth + 16

WheelHandler {
    acceptedModifiers: Qt.ControlModifier
    onWheel: function(event) {
        libraryRoot.adjustThumbnailScale(event.angleDelta.y > 0 ? 0.08 : -0.08)
        event.accepted = true
    }
}
```

### 右下角控制器

修改前：无缩略图尺寸控制入口。

修改后：新增 `thumbnailSizeControl`，包含半透明 2x2 网格图标和 `ThemedSlider`，范围为 `0.85` 到 `1.45`。
