## 代码前后对比

### 素材管理中心选择接口

修改前：


### 逐帧搜索展示

修改前：`selectedFrames()` 返回当前视频全部逐帧记录。

修改后：

```cpp
const auto terms = searchTerms(m_searchText);
for (const auto &frame : m_detail.frames) {
    if (!frameMatches(frame, terms)) {
        continue;
    }
    frames.append(QVariantMap{
        {QStringLiteral("matchText"), matches.join(QStringLiteral("、"))}
    });
}
```

### SQL 搜索范围

修改前：搜索主要依赖 `video_search_fts`、`search_text`、文件名、项目名和素材源名。

修改后：增加 `EXISTS` 查询 `video_frame_analysis`，覆盖 `caption`、`tags_json`、`objects_json`、`actions`、`setting_text`。

### 摘要 prompt

修改前：摘要要求为 1-2 句。

修改后：摘要要求为 4-6 句，覆盖主体、场景、动作变化、可见物体、氛围和镜头内容变化；输出结构仍保持 `{summary, keywords, scenes}`。
