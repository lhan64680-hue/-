# 107-自动更新支持系统代理

日期：2026-06-27

## 已完成内容

- 已为 GitHub 更新检查增加系统代理通道，优先读取系统代理并显式传给 `curl.exe`。
- 已为更新包下载增加系统代理通道，和版本检查共用同一套“代理优先、直连回退”策略。
- 已保留原有直连能力，当系统代理不可用或请求失败时会自动回退直连，不阻塞应用启动。
- 已补充代理相关单测，覆盖代理 URL 生成与代理列表选择。

## 具体修改的代码前后对比

### 更新检查从单一直连改为双通道

修改前：

```cpp
m_checkProcess->setArguments({
    QStringLiteral("--noproxy"),
    QStringLiteral("*"),
    ...
    QString::fromLatin1(kLatestReleaseUrl)
});
```

修改后：

```cpp
const auto proxyUrl = systemProxyUrl();
launchCheckProcess(proxyUrl, !proxyUrl.isEmpty());
```

### 下载更新包从单一直连改为双通道

修改前：

```cpp
m_downloadProcess->setArguments({
    QStringLiteral("--noproxy"),
    QStringLiteral("*"),
    ...
    release.installerUrl
});
```

修改后：

```cpp
const auto proxyUrl = systemProxyUrl();
launchDownloadProcess(proxyUrl, !proxyUrl.isEmpty());
```

修改前：

```cpp
QCOMPARE(UpdateService::proxyUrlForNetworkProxy(proxy), QStringLiteral("http://127.0.0.1:7890"));

```
