## 二、本次发布内容

- 新增 Flow 风格 Windows 全局快捷搜索窗口，默认快捷键 `Alt+Space`。
- 复用自然语言、FTS5、BGE、结构化查询和视觉模型搜索链路。
- 搜索结果按文件夹、画面帧和素材分区显示。
- 新增快捷搜索启停、自定义快捷键、冲突提示、托盘驻留与开机启动设置。
- 移除本地视觉模型的每日调用上限、请求扣减和次数显示。
- 快捷搜索调整为软件一致的中性深灰主题。
- 新增真实全局热键、设置持久化和 UI 契约测试。

## 三、代码前后对比

```text
修改前：
- 只有主窗口搜索框，没有独立快捷搜索窗口。
- 没有 RegisterHotKey、托盘入口或快捷键设置项。
- 本地视觉模型仍受每日额度、计数和扣减逻辑限制。
- 快捷搜索候选样式使用彩色强调色。

修改后：
- Alt+Space 在任意窗口拉起独立搜索框。
- 支持键盘选择、打开、定位和 Esc 收起。
- 本地视觉模型只受功能开关、隐私授权和模型配置约束，不限调用次数。
- 窗口使用固定中性深灰灰阶色板。
```
