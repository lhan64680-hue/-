### 立即更新从静默自更新改为打开安装包

修改前：

```cpp
scriptLines << QStringLiteral("    $installerArgs = @('/VERYSILENT', '/NORESTART', '/SUPPRESSMSGBOXES', $installerDirArg)")
            << QStringLiteral("    $process = Start-Process -FilePath $installerPath -ArgumentList $installerArgs -Verb RunAs -Wait -PassThru")
            << QStringLiteral("    if ($process.ExitCode -eq 0 -and Test-Path $appExe) {")
            << QStringLiteral("        Start-Process -FilePath $appExe | Out-Null")
            << QStringLiteral("    }");
```

修改后：

```cpp
scriptLines << QStringLiteral("    $installerLogArg = '/LOG=\"' + $installerUiLogPath + '\"'")
            << QStringLiteral("    $installerArgs = @($installerDirArg, $installerLogArg)")
            << QStringLiteral("    $process = Start-Process -FilePath $installerPath -ArgumentList $installerArgs -Verb RunAs -Wait -PassThru")
            << QStringLiteral("    if ($process.ExitCode -eq 0) {")
            << QStringLiteral("        Write-UpdateLog '更新安装包已正常结束，请按安装向导完成升级。'")
            << QStringLiteral("    }");
```
