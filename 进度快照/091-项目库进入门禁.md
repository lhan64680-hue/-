## 代码前后对比

### 1. 新增项目进入态

修改前：

```cpp
Q_PROPERTY(QVariantList workspaceTabs READ workspaceTabs CONSTANT)
```

修改后：

```cpp
Q_PROPERTY(bool projectEntered READ projectEntered NOTIFY stateChanged)
Q_PROPERTY(QVariantList workspaceTabs READ workspaceTabs NOTIFY stateChanged)
```

### 2. 未进入项目时禁止切页

修改前：

```cpp
m_currentWorkspace = normalizedValue;
emit currentWorkspaceChanged();
```

修改后：

```cpp
if (normalizedValue != WorkspaceId::ProjectLibrary && !projectEntered()) {
    m_lastMessage = "请先在项目库点击项目卡片进入项目。";
    m_currentWorkspace = WorkspaceId::ProjectLibrary;
    emit stateChanged();
    return;
}
```

### 3. 顶部标签置灰

修改前：

```qml
onClicked: shellVm.currentWorkspace = modelData.value
```

修改后：

```qml
enabled: tabEnabled
opacity: tabEnabled ? 1.0 : 0.44
onClicked: if (tabEnabled) shellVm.currentWorkspace = modelData.value
```

### 4. 主界面可操作判断

修改前：

```qml
readonly property bool hasOpenProject: root.shellViewModel && root.shellViewModel.projectPath.length > 0
```

修改后：

```qml
readonly property bool hasOpenProject: root.shellViewModel && root.shellViewModel.projectEntered
```
