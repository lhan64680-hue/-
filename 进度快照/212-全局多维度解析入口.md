### ViewModel 全局入口

新增：

```cpp
Q_PROPERTY(bool canAnalyzeVisibleDimensions READ canAnalyzeVisibleDimensions NOTIFY dimensionAnalysisChanged)
Q_INVOKABLE void analyzeVisibleDimensions(const QVariantList &dimensions);
```

新增逻辑：

```cpp
for (const auto &asset : m_assets) {
    const auto state = m_dimensionProgressByVideoKey.value(asset.videoKey);
    if (!canAnalyzeAsset(asset)
        || asset.analysisStatus != VideoAnalysisStatus::Ready
        || state.running) {
        continue;
    }

    QString errorMessage;
    if (m_analysisService->analyzeDimensions(asset.videoKey, dimensionNames, &errorMessage)) {
        ++accepted;
    }
}
```
