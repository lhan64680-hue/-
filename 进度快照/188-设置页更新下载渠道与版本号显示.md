## 具体修改的代码前后对比

修改前：设置页只有检查更新按钮，不显示当前版本号，不支持选择更新下载渠道。

```qml
Text {
    text: "设置"
}

ActionButton {
    text: viewModel && viewModel.updateBusy ? "检查中..." : "检查更新"
    onClicked: if (viewModel) {
        viewModel.checkForUpdates()
    }
}
```

修改后：设置页显示当前版本号，并可配置下载更新渠道。

```qml
Text {
    text: viewModel ? viewModel.currentVersionLabel : "当前版本：v0.0.0"
}

ThemedComboBox {
    model: [
        { label: "自动检测代理", value: 0 },
        { label: "手动代理", value: 1 },
        { label: "直连下载", value: 2 }
    ]
}
```

配置层新增：

```cpp
int updateDownloadMode() const;
void setUpdateDownloadMode(int value);

QString updateManualProxyUrl() const;
void setUpdateManualProxyUrl(const QString &value);
```
