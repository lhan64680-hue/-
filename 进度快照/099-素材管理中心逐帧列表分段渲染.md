### 2. 逐帧结果改为首批渲染

修改前：

```cpp
for (const auto &frame : m_detail.frames) {
    ...
    m_selectedFramesCache.append(QVariantMap{ ... });
}
```

修改后：

```cpp
constexpr int kInitialVisibleFrameCount = 24;

for (const auto &frame : m_detail.frames) {
    ...
    m_selectedAllFramesCache.append(QVariantMap{ ... });
}

applySelectedFrameExpansion();
```

并新增：

```cpp
void MaterialCenterViewModel::applySelectedFrameExpansion()
{
    m_selectedFramesCache.clear();

    if (m_selectedFramesExpanded || m_selectedAllFramesCache.size() <= kInitialVisibleFrameCount) {
        m_selectedFramesCache = m_selectedAllFramesCache;
        return;
    }

    for (int index = 0; index < kInitialVisibleFrameCount && index < m_selectedAllFramesCache.size(); ++index) {
        m_selectedFramesCache.append(m_selectedAllFramesCache.at(index));
    }
}
```

### 3. 右侧逐帧区增加数量提示与展开全部

修改前：

```qml
Text {
    Layout.fillWidth: true
    visible: viewModel && viewModel.hasSelection && viewModel.selectedFrameSearchStatus.length > 0
    text: viewModel ? viewModel.selectedFrameSearchStatus : ""
}

Repeater {

```

修改后：

```qml
Text {
    Layout.fillWidth: true
    visible: viewModel && viewModel.hasSelection && viewModel.selectedFrameSearchStatus.length > 0
    text: viewModel ? viewModel.selectedFrameSearchStatus : ""
}

RowLayout {
    Layout.fillWidth: true
    visible: viewModel
        && viewModel.hasSelection
        && !viewModel.selectedFramesLoading
        && viewModel.selectedFrameCount > 0

    Text {
        Layout.fillWidth: true
        text: !viewModel ? ""
            : (viewModel.selectedVisibleFrameCount < viewModel.selectedFrameCount
                ? "当前仅渲染前 " + viewModel.selectedVisibleFrameCount + " / " + viewModel.selectedFrameCount + " 帧"
                : "当前显示 " + viewModel.selectedVisibleFrameCount + " 帧")
    }

    ActionButton {
        visible: viewModel && viewModel.canExpandSelectedFrames
        text: viewModel && viewModel.selectedFramesExpanded ? "收起" : "展开全部"
        onClicked: if (viewModel) viewModel.toggleSelectedFramesExpanded()
    }
}

Repeater {
    model: viewModel ? viewModel.selectedFrames : []
}
```
