# 118-视觉返回自动修复请求

日期：2026-06-29

## 已完成内容

- 已读取上一轮最新快照：`117-视觉返回字段归一化.md`。
- 已创建阶段备份：`backup/v0.1.100`。
- 已完成模块三：视觉返回自动修复请求。
- `VisionResponseParser` 新增 `extractAssistantContent()`，可以从 OpenAI-compatible chat 响应中提取原始 assistant 文本。
- `VisionApiClient` 新增一次性 `repairAssistantPayload()` 修复流程。
- 帧解析首次 JSON 解析失败时，会把原始返回内容发送给模型，要求转换为 `{"caption":"","tags":[],"objects":[],"actions":"","setting":""}`。
- 视频汇总首次 JSON 解析失败时，会把原始返回内容发送给模型，要求转换为 `{"summary":"","keywords":[],"scenes":[]}`。
- 如果首次 JSON 可解析但归一化后为空，也会触发同一轮自动修复。
- 自动修复只尝试一次，不循环重试；修复仍失败时继续返回明确失败原因。
- 自动修复提示词明确要求只转换格式、不新增事实、不解释、不返回 Markdown。
- 已补充单测覆盖纯文本、CSV、Markdown 表格原始内容提取。
- 已更新 `docs/素材管理.md` 的视觉接口说明。
- 已新增模块溯源文档：`backup/v0.1.100/视觉返回自动修复请求-前后对比.md`。

## 当前修改到哪个模块

当前完成模块：模块三，自动修复请求。

本模块解决“模型返回了有用内容但不是 JSON”的一类问题；如果自动修复也失败，当前仍会失败，纯文本兜底落库留到模块四处理。

## 具体修改的代码前后对比

### 原始内容提取

修改前：

```cpp
std::optional<QJsonObject> parseAssistantJson(const QByteArray &responseBody, QString *errorMessage);
```

修改后：

```cpp
std::optional<QString> extractAssistantContent(const QByteArray &responseBody, QString *errorMessage);
std::optional<QJsonObject> parseAssistantJson(const QByteArray &responseBody, QString *errorMessage);
```

### 帧解析自动修复

修改前：

```cpp
auto payload = VisionResponseParser::parseAssistantJson(result.body, errorMessage);
if (!payload.has_value()) {
    return std::nullopt;
}

return VisionResponseParser::normalizeFrameAnalysis(*payload, errorMessage);
```

修改后：

```cpp
auto payload = VisionResponseParser::parseAssistantJson(result.body, &parseError);
if (!payload.has_value()) {
    payload = repairAssistantPayload(result.body, endpoint, apiKey, model, timeoutSec, schema, parseError, 350, errorMessage);
}

auto analysis = VisionResponseParser::normalizeFrameAnalysis(*payload, &normalizeError);
if (!analysis.has_value() && !usedRepair) {
    payload = repairAssistantPayload(result.body, endpoint, apiKey, model, timeoutSec, schema, normalizeError, 350, errorMessage);
}
```

### 视频汇总自动修复

修改前：

```cpp
auto payload = VisionResponseParser::parseAssistantJson(result.body, errorMessage);
if (!payload.has_value()) {
    return std::nullopt;
}

return VisionResponseParser::normalizeVideoSummary(*payload, errorMessage);
```

修改后：

```cpp
auto payload = VisionResponseParser::parseAssistantJson(result.body, &parseError);
if (!payload.has_value()) {
    payload = repairAssistantPayload(result.body, endpoint, apiKey, model, timeoutSec, schema, parseError, 900, errorMessage);
}

auto summary = VisionResponseParser::normalizeVideoSummary(*payload, &normalizeError);
if (!summary.has_value() && !usedRepair) {
    payload = repairAssistantPayload(result.body, endpoint, apiKey, model, timeoutSec, schema, normalizeError, 900, errorMessage);
}
```
