### 同步服务

修改前：

```cpp
if (m_syncRunning.exchange(true) || !m_globalDatabaseManager || !m_projectService || !m_projectService->hasOpenProject()) {
    return;
}
```

修改后：

```cpp
if (!m_globalDatabaseManager || !m_projectService || !m_projectService->hasOpenProject()) {
    return;
}
if (m_syncRunning.exchange(true)) {
    m_syncPending = true;
    return;
}
m_syncPending = false;
```

新增同步结束补跑：

```cpp
void MaterialCatalogSyncService::finishSyncRun()
{
    m_syncRunning = false;
    if (m_syncPending.exchange(false)) {
        QMetaObject::invokeMethod(this, &MaterialCatalogSyncService::syncCurrentProject, Qt::QueuedConnection);
    }
}
```

### 文件筛选下拉

修改前：

```cpp
m_assetTypeOptions = m_queryService->fetchAssetTypeOptions();
m_assetTypeOptions.prepend(QVariantMap{{QStringLiteral("value"), -1}, {QStringLiteral("label"), QStringLiteral("全部类型")}});
```

修改后：

```cpp
m_assetTypeOptions = fileTypeOptions();
```

`fileTypeOptions()` 固定返回“全部文件”和所有文件类型，避免只根据当前数据库已有类型生成下拉。

### QML 当前筛选状态

修改前：

```qml
ThemedComboBox {
    Layout.preferredWidth: 126
    model: viewModel ? viewModel.assetTypeOptions : []
    textRole: "label"
    onActivated: if (viewModel) viewModel.setAssetTypeFilter(model[index].value)
}
```

修改后：

```qml
ThemedComboBox {
    Layout.preferredWidth: 136
    model: viewModel ? viewModel.assetTypeOptions : []
    textRole: "label"
    currentIndex: viewModel ? root.optionIndex(viewModel.assetTypeOptions, viewModel.assetTypeFilter) : 0
    onActivated: if (viewModel) viewModel.setAssetTypeFilter(model[index].value)
}
```
