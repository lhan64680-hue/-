## 具体修改的代码前后对比

### 启动自动检查

修改前：

```qml
Component.onCompleted: root.forceActiveFocus()
```

修改后：

```qml
Component.onCompleted: {
    root.forceActiveFocus()
    Qt.callLater(function() {
        if (root.settingsViewModel) {
            root.settingsViewModel.beginStartupUpdateFlow()
        }
    })
}
```

## 验证结果

- 启动自动检查、手动检查、下载完成确认、待安装复用共用同一套 `UpdateService + SettingsViewModel` 流程。
- 选择 `下次启动更新` 后不会清除待安装状态；下次启动会优先弹出同一确认框。
- 选择 `立即更新` 后会写出 PowerShell 更新脚本并调用静默安装参数：
  - `/VERYSILENT`
  - `/NORESTART`
  - `/SUPPRESSMSGBOXES`
