## 具体修改的代码前后对比

### 基础解析进度触发多维度按钮刷新

修改前：

```cpp
if (videoKey == m_detail.asset.videoKey) {
    emit analysisProgressChanged();
}
```

修改后：

```cpp
if (videoKey == m_detail.asset.videoKey) {
    emit analysisProgressChanged();
    emit dimensionAnalysisChanged();
}
```

### 基础解析运行中禁用多维度解析

修改前：

```cpp
return hasSelection()
    && m_detail.asset.analysisStatus == VideoAnalysisStatus::Ready
    && !selectedDimensionAnalysisBusy();
```

修改后：

```cpp
return hasSelection()
    && m_detail.asset.analysisStatus == VideoAnalysisStatus::Ready
    && !selectedAnalysisBusy()
    && !selectedDimensionAnalysisBusy();
```

### 多维度解析弹窗响应式尺寸

修改前：

```qml
width: 560
height: 560
x: Math.round((root.width - width) / 2)
y: Math.round((root.height - height) / 2)
```

修改后：

```qml
width: Math.max(280, Math.min(560, root.width - 48))
height: Math.max(360, Math.min(560, root.height - 48))
x: Math.max(12, Math.round((root.width - width) / 2))
y: Math.max(12, Math.round((root.height - height) / 2))
```
