# 进度快照 248：视觉接口 502 根因定位与客户端容错部署完成

## 已完成内容

### 根因定位

- 截图中的自然语言本地解析与语义检索均正常，失败点仅在视觉模型辅助请求。
- 当前项目日志 `D:/data/影视管家/002/logs/app.log` 在 2026-07-14 20:59:19 至 21:00:38 连续记录 13 次空响应体 502。
- 当前视觉服务为 OpenAI-compatible `/v1/chat/completions` 接口。
- 远端端口可建立 TCP 连接，但直连时不返回 HTTP 数据；Windows 系统代理 `127.0.0.1:7890` 将该上游异常转换为 502。
- `/v1/models` 和 95 字节最小纯文本请求同样失败，排除图片大小、编码、JSON Schema、查询内容与客户端超时。
- 远端 SSH 在线但本机密钥均无匹配账户；已保存的远程桌面凭据也无法建立会话，因此未能从本机重启远端模型服务。

### 客户端修复

- `VisionApiClient.cpp` 新增 500/502/503/504 专用错误文案。
- 对瞬时 HTTP 服务故障及非超时网络中断最多重试一次，退避 350ms。
- 持续失败时严格停止在两次请求。
- 日志新增 `attempt`、`elapsed_ms`、响应体摘要与错误摘要。
- 重试成功时记录瞬时故障恢复日志。

关键前后对比：

```text
修改前：失败一次后直接返回“视觉接口返回异常状态码：502”
修改后：首次 502 -> 350ms 退避 -> 再试一次
        成功则继续；仍失败则返回“视觉模型网关无法连接后端（502），已重试 1 次仍失败”
```
