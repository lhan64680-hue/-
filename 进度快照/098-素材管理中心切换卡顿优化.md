## 具体修改的代码前后对比

### 1. 选中素材改为延后加载重明细

修改前：

```cpp
void MaterialCenterViewModel::selectVideo(const QString &videoKey)
{
    if (m_detail.asset.videoKey == videoKey.trimmed()) {
        return;
    }
    m_detail = {};
    m_detail.asset.videoKey = videoKey.trimmed();
    refreshDetail();
    emit selectionChanged();
    emit analysisProgressChanged();
}
```

修改后：

```cpp
void MaterialCenterViewModel::selectVideo(const QString &videoKey)
{
    const auto normalizedKey = videoKey.trimmed();
    if (m_detail.asset.videoKey == normalizedKey) {
        return;
    }
    prepareSelection(normalizedKey);
    emit selectionChanged();
    emit analysisProgressChanged();
}
```

### 3. 详情区增加加载态与异步图片加载

修改前：

```qml
Image {
    anchors.fill: parent
    fillMode: Image.PreserveAspectCrop
    source: viewModel ? viewModel.selectedThumbnailUrl : ""
}

Text {
    visible: viewModel && viewModel.hasSelection && viewModel.selectedFrames.length === 0
    text: "当前还没有逐帧解析结果。"
}
```

修改后：

```qml
Image {
    anchors.fill: parent
    fillMode: Image.PreserveAspectCrop
    asynchronous: true
    cache: true
    source: viewModel ? viewModel.selectedThumbnailUrl : ""
}

Text {
    anchors.centerIn: parent
    visible: !viewModel || !viewModel.hasSelection || viewModel.selectedThumbnailUrl.toString().length === 0
    text: viewModel && viewModel.selectedFramesLoading ? "正在加载预览..." : "暂无多宫格拼图"
}

Text {
    visible: viewModel
        && viewModel.hasSelection
        && !viewModel.selectedFramesLoading
        && viewModel.selectedFrames.length === 0
    text: "当前还没有逐帧解析结果。"
}
```
