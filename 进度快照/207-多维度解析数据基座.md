## 具体修改的代码前后对比

详细阶段对比将在 `backup/v0.1.160/多维度解析功能-代码对比.md` 中汇总。

### 详情数据结构

修改前：

```cpp
struct VideoAnalysisDetail {
    GlobalVideoAsset asset;
    QVector<FrameAnalysisRecord> frames;
};
```

修改后：

```cpp
struct MaterialDimensionAnalysis {
    QString name;
    QString detail;
    QString analyzedAt;
};

struct VideoAnalysisDetail {
    GlobalVideoAsset asset;
    QVector<FrameAnalysisRecord> frames;
    QVector<MaterialDimensionAnalysis> dimensionAnalyses;
};
```

### 数据库表

修改前：全局库只有基础摘要、逐帧解析和解析任务表。

修改后：

```sql
CREATE TABLE IF NOT EXISTS material_dimension_analysis (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    video_key TEXT NOT NULL,
    dimension_key TEXT NOT NULL,
    dimension_name TEXT NOT NULL,
    detail TEXT NOT NULL DEFAULT '',
    model_name TEXT NOT NULL DEFAULT '',
    prompt_version TEXT NOT NULL DEFAULT '',
    analyzed_at TEXT NOT NULL DEFAULT '',
    FOREIGN KEY(video_key) REFERENCES global_video_asset(video_key) ON DELETE CASCADE,
    UNIQUE(video_key, dimension_key)
);
```

### 搜索补充

修改前：搜索只覆盖摘要、正文、技术摘要、文件信息和逐帧解析。

修改后：额外通过 `material_dimension_analysis.dimension_name/detail` 命中素材。
